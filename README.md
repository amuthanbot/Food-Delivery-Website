# 🍔 Foodiees – Order Food Online

> **Delivering happiness, one meal at a time.** A modern, fully responsive food delivery frontend web application built with pure HTML, CSS, and JavaScript — no frameworks, no backend required.

---

## 📸 Overview

Foodiees is a feature-rich food delivery website inspired by platforms like **Swiggy** and **Zomato**. It offers a beautiful, animated UI with a complete menu browsing and cart management experience — all powered by vanilla web technologies.

---

## ✨ Features

### 🏠 Home Page
- **Hero Section** – Eye-catching landing area with delivery stats (500+ restaurants, 50k+ customers, 30-min delivery) and an address search bar
- **Category Browser** – Filter food by cuisine: Indian 🍛, Fast Food 🍔, Italian 🍕, Japanese 🍣, Desserts 🎂, Drinks 🥤
- **Offers & Deals Banner** – Highlights active promotions (40% off first order, free delivery, daily combos)
- **Food Menu Grid** – Dynamic card-based layout with food images, ratings, delivery time, veg/non-veg badges, and bestseller labels
- **Why Choose Us Section** – Feature highlights: Lightning Fast, Top Quality, Easy Payment, Safe & Hygienic

### 🔍 Search & Filter
- **Live Search** – Real-time filtering by food name, description, or category
- **Category Filter** – Click any category card to filter the menu instantly
- **Sort Options** – Sort by Price (Low→High / High→Low) or Top Rated

### 🛒 Cart System
- **Add / Remove Items** – Add items directly from the menu grid
- **Quantity Control** – Increase or decrease quantity from both the menu and cart page
- **Persistent Cart** – Cart is saved in `localStorage` and survives page refreshes
- **Order Summary** – Breakdown of subtotal, delivery fee (free above ₹299), and 5% taxes
- **Promo Code** – Apply code `FOODIEES10` for a discount
- **Checkout Flow** – One-click order placement with a success toast notification

### 📱 Responsive Design
- **Mobile Navigation** – Hamburger menu with animated slide-in drawer
- **Responsive Grid** – Food cards adapt from 1 to 4 columns based on screen size
- **Touch-Friendly** – All buttons and controls optimized for mobile

### 🎨 UI / UX
- Sticky navbar with scroll-aware background blur
- Smooth CSS animations and staggered card entry effects
- Toast notifications for all user actions
- Dark-themed premium design with gradient accents

## 🚀 Getting Started

No installation or build step required! Just open the project in a browser.

### Option 1 — Direct Open
Double-click `index.html` to open it in your default browser.

### Option 2 — Live Server (Recommended)
For the best experience (avoids potential CORS issues with local images):

1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. The site opens at `http://127.0.0.1:5500`
